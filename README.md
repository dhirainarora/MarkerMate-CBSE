# 🚀 MarkerMate CBSE – AI Answer Checker

## 📌 Overview

MarkerMate CBSE is an AI-powered web application that helps Class 10 students practice and improve their board exam answers using CBSE-style evaluation.

Students can submit answers (typed or via image), and the system evaluates them using structured rubric-based marking — just like a real examiner.

---

## 🌐 Live Demo

👉 https://marker-mate-cbse-98c9.vercel.app

---

## 🎯 Problem It Solves

Students often struggle with:

* Not knowing how answers are marked in board exams
* Lack of detailed feedback from teachers
* Difficulty improving answer-writing skills

MarkerMate solves this by providing **instant, structured, and actionable feedback**.

---

## ✨ Key Features

### 🧠 AI-Based Answer Evaluation

* Evaluates answers using CBSE-style rubrics
* Assigns marks based on weightage (1,2,3,5 marks)

### 📸 OCR Support

* Upload handwritten or printed answers
* Automatically extracts text for evaluation

### 📊 Detailed Feedback System

* Marks awarded
* Examiner-style justification
* Missing points
* Improvement suggestions
* Ideal answer approach
* Rewritten sample answer

### 📁 Student Dashboard

* View past submissions
* Filter by subject and date
* Revisit feedback anytime

## 🏗️ Tech Stack

* **Frontend:** Next.js + TypeScript
* **Backend:** API Routes (Next.js)
* **Database:** PostgreSQL
* **ORM:** Prisma
* **AI/LLM:** Gemini / OpenAI
* **OCR:** Image-to-text processing

---

## ⚙️ How It Works

1. Student selects subject
2. Enters question + answer (or uploads image)
3. Selects mark weightage
4. AI evaluates using rubric system
5. Returns marks + detailed feedback

---

## 🧪 Example Output

* Awarded Marks: 3/5
* Missing Points: Key concept explanation
* Suggestion: Add structured points with examples
* Rewritten Answer: Provided
